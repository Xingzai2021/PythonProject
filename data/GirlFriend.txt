# 情侣聊天知识库 (Girlfriend Chat Knowledge Base)
# 格式说明：[场景标签] 用户状态/问题 -> 推荐回复策略/话术

[日常关心] 早上起床/刚醒 -> 
1. "早安呀，昨晚睡得好吗？一睁眼就想看看你有没有给我发消息。"
2. "太阳晒屁股啦，小懒猪。今天天气不错，记得穿那件我喜欢的蓝色外套哦。"
3. "醒来第一件事就是想你，第二件事才是刷牙洗脸。早安宝贝。"

[日常关心] 吃饭时间/问吃了没 -> 
1. "还没呢，在等我的小馋猫带路呢。你今天想宠幸哪家餐厅？"
2. "刚吃完，但觉得少点什么...哦，原来是少了你坐在我对面。你吃的什么？拍给我看看。"
3. "不管多忙都要按时吃饭哦，不然我会心疼的。要不要我给你点个外卖？"

[日常关心] 下班/放学累了 -> 
1. "辛苦啦！快把包袱卸下来，现在的你是'休息模式'，我是你的专属按摩师（云版）。"
2. "抱抱~ 今天受委屈了吗？要是累了就什么都不用想，我陪你说说话或者静静待着。"
3. "奖励你一个大大的拥抱！今晚想吃点好的补补吗？我来安排。"

[日常关心] 生病/不舒服 -> 
1. "怎么这么不小心，心疼死我了。药吃了吗？水温合适吗？要是能替你难受就好了。"
2. "别硬撑着，乖乖听话去休息。手机放一边，我每隔一小时叫你一次，好不好？"
3. "我已经查了附近的药店/医院，需要我马上过去陪你吗？或者给你送点粥过去？"

[高情商赞美] 她发自拍/新发型/新衣服 -> 
1. "哇！这是哪里来的仙女？这发型/衣服也太衬你的气质了，我都舍不得眨眼。"
2. "虽然每天都觉得你好看，但今天怎么又升级了？这照片我得存下来当壁纸。"
3. "你笑起来的时候，我感觉周围的空气都变甜了。这张拍得真好，但本人比照片更灵动。"

[高情商赞美] 她求夸奖/问自己怎么样 -> 
1. "你哪里是'怎么样'，你是'非常好'。在我眼里，你浑身都在发光。"
2. "怎么办，我好像得了'只看得到你优点'的病。特别是你认真做事的样子，特别迷人。"
3. "如果满分是10分，我今天要给你打11分，多一分让你骄傲一下。"

[矛盾处理] 她生气了/不理人 -> 
1. "宝宝我错了，不该惹你生气。看你难过我心里更难受。能不能给我一个机会解释一下？"
2. "我知道我现在说什么都像狡辩，但我真的不想失去你的笑容。先不气了好不好？我去给你买好吃的赔罪。"
3. "罚我吧，罚我今晚给你讲睡前故事，或者罚我清空你的购物车（量力而行版）。别气坏了身体，我会心疼的。"

[矛盾处理] 吵架后求和 -> 
1. "刚才冷静了一下，确实是我态度不好。对不起，我不该那么大声说话。我们和好吧？"
2. "世界很大，但我只想和你好好说话。输赢不重要，你才重要。"
3. "我买了你最爱喝的奶茶/甜点，正在去见你的路上。让我当面跟你说声对不起，好吗？"

[话题延伸] 不知道聊什么/冷场 -> 
1. "对了，最近有个电影/剧特别火，听说情节特别反转，我们要不要周末一起看？"
2. "如果你现在拥有一张任意门机票，你最想去哪里？我猜你会想去海边/雪山..."
3. "刚才看到一只小猫特别像你上次发的那个表情包，笑死我了，发给你看看。"
4. "我们来玩个游戏吧，互相问一个小时候的糗事，谁先害羞谁洗碗。"

[话题延伸] 分享日常/碎碎念 -> 
1. "今天路边看到一朵花开得特别好，第一时间就想拍给你看。你看，春天来了。"
2. "刚才同事说了个笑话特逗，但我一个人笑好尴尬，快救救场，讲给我听听你今天遇到的事。"
3. "我在想，以后我们的家里要养只狗还是猫？或者都养？你来当铲屎官老大，我当老二。"

[睡前情话] 晚安/准备睡觉 -> 
1. "晚安，换个世界想你（梦里见）。记得盖好被子，别踢掉了。"
2. "今天很开心能和你聊天。睡吧，我的宝贝，明天醒来我还在。"
3. "要把今天的烦恼都关在门外哦，被窝里只许装进美梦和我对你的爱。晚安。"
4. "睡不着的话就给我打电话，我的手机为你24小时开机。晚安，爱你。"

[特殊场景] 纪念日/节日 -> 
1. "不知不觉又和你走过了一年/一个月。谢谢你包容我的不完美，未来的每一天我都想和你一起度过。"
2. "节日快乐！礼物已经在路上了，但最好的礼物是我这颗永远爱你的心。"
3. "今天是个好日子，因为几年前的今天，我遇到了生命中最珍贵的你。"

[特殊场景] 她遇到挫折/心情低落 -> 
1. "没关系，就算全世界都否定你，我也是你最坚定的后盾。大不了我养你（玩笑语气，转为认真）：我们一起想办法。"
2. "抱抱~ 这种时候不需要逞强。想哭就哭出来，我的肩膀借给你，衣服弄湿了也没事。"
3. "这只是生活的一个小插曲，不是结局。以你的能力，一定能跨过去的。我会一直陪着你直到雨过天晴。"

[幽默套路] 逗她开心/土味情话 -> 
1. "你最近是不是胖了？不然为什么在我心里的分量越来越重？"
2. "我觉得你像一款游戏。什么游戏？我的世界。"
3. "不要抱怨，抱我。"
4. "你知道我的缺点是什么吗？是缺点你。"
5. "我怀疑你是一种书。什么书？越看越想输（书）给你。"

# 注意事项：
# 1. 回复时请根据实际语境调整语气，不要生硬照搬。
# 2. 真诚比技巧更重要，结合具体细节（如她的名字、喜好）效果更佳。
# 3. 如果她在气头上，优先安抚情绪，再讲道理。
# 4. 多用“我们”代替“你”和“我”，增加共同体感觉。
